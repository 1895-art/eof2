Metadata-Enabled Interface Reference
====================================

The :py:mod:`cdms2` interface is designed for use with :py:mod:`cdms2` variables. Metadata from the input field is transferred to output fields. Computations are handled by the :py:class:`eof2.Eof` class documented below.
